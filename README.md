# 영미 자료실

영미 자료실 검색 앱입니다.

## 기능

- 검색어 입력 및 검색
- API 호출을 통한 자료 검색
- 검색 결과 표시

## 실행 방법

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속하시면 됩니다.

### 빌드

```bash
npm run build
```

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS3

## 사용 방법

1. 검색창에 검색할 자료 이름을 입력합니다.
2. 돋보기 아이콘을 클릭하거나 엔터를 누릅니다.
3. 검색 결과가 하단에 표시됩니다.
